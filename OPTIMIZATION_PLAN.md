# Leeelicspace 博客优化实施计划

## 📋 项目概述

将个人博客从当前基础设计升级为现代化、优雅的界面，同时添加中英文双语支持。

---

## 🎯 Phase 1: UI 基础优化 (当前阶段)

### 1.1 设计系统升级

**配色方案简化**
- Light: 纯白背景 + 深灰文字 + 蓝色强调
- Dark: 深蓝灰背景 + 浅灰文字 + 青色强调
- 移除复杂的 Catppuccin 主题，改为专业双色方案

**字体系统**
- 标题: Inter (英文) + Noto Sans SC (中文)
- 正文: 系统字体栈，保证加载速度
- 代码: JetBrains Mono / Fira Code

**间距系统**
- 使用 4px 基准网格
- 增加页面留白，提升呼吸感
- 内容区域最大宽度: 768px (文章), 1200px (首页)

## 🌍 Phase 2: 双语支持 (国际化)

### 2.1 技术选型
- **库**: `next-intl` (最佳 App Router 支持)
- **语言**: 中文 (zh) / 英文 (en)
- **默认**: 根据用户浏览器语言自动检测

### 2.2 实施步骤
1. 安装 next-intl
2. 配置 middleware 路由拦截
3. 创建语言文件 (zh.json / en.json)
4. 抽离所有硬编码文本
5. 添加语言切换组件
6. 支持 URL 路径语言标识 (可选)

### 2.3 需要翻译的内容
- 所有 UI 文本 (导航、按钮、标签)
- 页面标题和描述
- 文章元数据 (日期格式、阅读时间)
- 错误提示信息
- RSS 描述
