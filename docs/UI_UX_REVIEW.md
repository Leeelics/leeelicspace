3. 动态路由参数处理错误

// 检查 app/[locale]/posts/[postId]/page.tsx
// 确保正确获取 postId 参数

## 🟠 中等问题

### 2. 登录页布局异常
**问题描述**：登录页面显示侧边栏导航，与登录状态不符

**截图证据**：`login-light.png`

**观察**：
- 左侧显示"总览、写作、文章管理"等导航项
- 未登录用户不应看到管理导航
- "退出登录"按钮出现在未登录状态

**修复建议**：
- 登录页使用独立布局，不包含侧边栏
- 或根据认证状态条件渲染侧边栏

### 3. 暗模式切换依赖 JavaScript
**问题描述**：使用 `prefers-color-scheme` 媒体查询无法自动切换主题

**观察**：
- 必须通过点击主题按钮才能切换
- 首次访问时不会跟随系统主题

**修复建议**：
```css
/* 在 globals.css 中添加 */
@media (prefers-color-scheme: dark) {
  :root:not([data-theme]) {
    /* 暗色变量 */
  }
}
```

### 4. 移动端首页布局问题
**问题描述**：移动端 (375px) 首页内容拥挤

**截图证据**：`homepage-mobile.png`

**具体问题**：
- 文章标题和标签换行混乱
- "按类别浏览"标签区域过于紧凑
- 搜索框宽度不足

**修复建议**：
- 移动端使用垂直堆叠布局
- 标签使用更小字号或省略显示
- 考虑移动端隐藏部分次要内容

## 🟡 轻微问题

### 5. 视觉层级不清晰

#### 5.1 首页 Hero 区域
**问题**：
- "leeelicspace" 大字标题与下方文章列表视觉连接不紧密
- "思考/探索/记录/分享" 背景文字干扰主内容阅读
- 日期"昨天"重复出现，占用空间

**建议**：
- 移除或淡化背景装饰文字
- 统一日期显示格式
- 添加更多视觉分隔

#### 5.2 标签样式不一致
**观察**：
- "全部"标签使用蓝色背景
- 其他标签使用灰色边框
- 激活状态不明确

**建议**：
- 使用一致的标签样式
- 明确区分激活/悬停状态

### 6. 颜色对比度问题

#### 6.1 暗模式文字颜色
**问题**：部分文字在暗色背景下对比度不足

**观察**：
- "阅读全文"链接颜色在深色背景不够突出
- 标签文字可能需要更亮的颜色

#### 6.2 链接颜色
**建议**：确保所有链接满足 WCAG AA 标准 (4.5:1 对比度)

### 7. 交互反馈缺失

#### 7.1 按钮悬停效果
**观察**：
- "返回首页"按钮无明显悬停反馈
- 文章卡片无悬停状态

**建议**：
- 添加微妙的阴影或颜色变化
- 卡片可添加轻微上浮效果

#### 7.2 加载状态
**问题**：文章详情页加载失败时无加载动画

**建议**：
- 添加骨架屏或加载指示器
- 错误页面提供刷新按钮

### 2. 暗模式优化

**当前暗模式问题**：
- 背景纯黑 (#000) 过于刺眼
- 建议使用深色灰 (#121212 或 #1a1a1a)

**推荐暗色调色板**：
```css
--bg-primary: #0f172a;      /* 深蓝灰背景 */
--bg-secondary: #1e293b;    /* 卡片背景 */
--text-primary: #f8fafc;    /* 主文字 */
--text-secondary: #94a3b8;  /* 次要文字 */
--accent: #3b82f6;          /* 强调色 */
```

### 3. 移动端适配

**优先级改进**：
1. 隐藏侧边栏（"按类别浏览"可折叠）
2. 文章列表改为卡片式
3. 搜索框全宽显示
4. 增加触摸目标尺寸 (最小 44px)

### 4. 无障碍改进

- 为所有图片添加 alt 属性
- 确保表单标签与输入框关联
- 添加跳过导航链接
- 支持键盘导航
